# JS-Logify
JS-Logify is a [bookmarklet](http://en.wikipedia.org/wiki/Bookmarklet) to convert logic expression in _similar_ [ASCII](http://en.wikipedia.org/wiki/Ascii) formatting to proper unicode logic runes.
